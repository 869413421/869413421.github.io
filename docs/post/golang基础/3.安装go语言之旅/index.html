golang基础(3.安装go语言之旅)

Feb 7, 2023

go语言之旅是官方提供的线上实践指南，教程涵盖了基础，方法接口，并发编程等大部分重要特性。鉴于网络问题下载安装到本地直接运行相对来说更便捷。

开启 go modules

go env -w GO111MODULE=on

下载运行

mkdir tour
cd tour
go mod init tour 
go get -u github.com/Go-zh/tour tour
